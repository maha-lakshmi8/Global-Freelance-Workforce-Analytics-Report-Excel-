# Global Freelance Workforce Analytics Report (Excel)

## Project Overview
The **Global Freelance Workforce Analytics Dashboard** is an interactive data analysis project built using **Microsoft Excel** to analyze freelancer demographics, skills distribution, experience levels, and earning patterns across different countries.

The dashboard transforms raw freelancer data into **meaningful insights through pivot tables, charts, and interactive filters**, enabling a deeper understanding of global freelance workforce trends.

This project demonstrates how Excel can be used for **data analysis, workforce analytics, and business intelligence reporting**.

---

## Project Objective

The main objectives of this project are:

- Analyze the **global distribution of freelancers**
- Identify **most common freelance skills**
- Evaluate **average freelancer ratings by skill**
- Understand **experience level distribution**
- Examine **hourly rate trends**
- Analyze freelancers by **country and language**
- Provide an **interactive dashboard for workforce insights**

---

## Dataset Information

The dataset contains detailed information about freelancers working globally.

### Dataset Columns

| Column | Description |
|------|-------------|
| Freelancer ID | Unique identifier for each freelancer |
| Full Name | Name of the freelancer |
| Gender | Gender of the freelancer |
| Age | Age of the freelancer |
| Country | Country where the freelancer is located |
| Language | Primary language spoken |
| Primary Skill | Main professional skill |
| Years of Experience | Total experience in years |
| Experience (Buckets) | Categorized experience groups |
| Hourly Rate | Freelancer hourly charge |
| Rating | Client rating score |
| Status | Freelancer activity status |
| Client Satisfaction | Customer satisfaction score |
